# math-puzzles-in-lean4

Math puzzles formalized in Lean 4,
using the Lean 3 repo https://github.com/dwrensha/math-puzzles-in-lean
as a starting point.

## building

Make sure you have `elan` installed, then do:

```
$ lake exe cache get
$ lake build
```
